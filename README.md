# The Scientific Calculator
This is the Scientific Calculator which is made with tkinter module in python.

It can do operations such as:
Primary Operations:
1. Addition
2. Subtraction
3. Multiplication
4. Division

Trigonometric Operations:
1. Sin
2. Cos
3. Tan
4. Sin inverse
5. Cos inverse
6. Tan inverse

Logarithmic Operations:
1. Log
2. Ln

Other Operations:
1. Factorial
2. Exponential (e)
3. Power (x^y)
4. Square root
